About Smart Contact Manager

Smart Contact Manager (SCM) is a user-friendly web-based application designed to help individuals securely store, manage, and organize personal contacts. Built using Spring Boot and Thymeleaf, it integrates modern UI/UX practices and secure session-based authentication to deliver a smooth and efficient experience.
